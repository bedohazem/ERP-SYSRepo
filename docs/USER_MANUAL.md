# ERP Store - دليل الاستخدام السريع

## 1. التثبيت
1. شغل ملف التثبيت ERP Store Setup.
2. اضغط Next.
3. بعد التثبيت افتح البرنامج من Desktop Shortcut.

## 2. تسجيل الدخول
بيانات الدخول الافتراضية:

Username: admin  
Password: 1234

يفضل تغيير كلمة المرور بعد أول تشغيل.

## 3. إضافة منتج
1. افتح شاشة المنتجات.
2. اضغط إضافة منتج.
3. اكتب اسم المنتج.
4. أضف Variant يحتوي على الباركود، المقاس، اللون، سعر الشراء، سعر البيع، والرصيد الافتتاحي.
5. احفظ المنتج.

## 4. البيع
1. افتح شاشة المبيعات.
2. ابحث بالباركود أو اسم المنتج.
3. أضف الكمية.
4. اختر العميل إذا كانت الفاتورة آجلة أو جزئية.
5. اكتب المدفوع.
6. احفظ الفاتورة واطبعها.

## 5. المرتجعات
1. افتح سجل الفواتير.
2. اختر الفاتورة.
3. اضغط مرتجع.
4. اختر الكمية المرتجعة.
5. احفظ المرتجع.
6. اطبع الفاتورة بعد المرتجع إذا احتجت.

## 6. المشتريات
1. افتح فواتير الشراء.
2. اختر المورد.
3. أضف الأصناف والكميات وسعر الشراء.
4. اكتب المدفوع.
5. احفظ الفاتورة.

## 7. العملاء والموردين
- العملاء: لإدارة بيانات العملاء والمديونيات.
- الموردين: لإدارة بيانات الموردين والمديونيات.

## 8. المخزون
شاشة المخزون تعرض:
- الكمية الحالية.
- الحد الأدنى.
- حالة الصنف.
- حركات المخزون.

يمكن عمل تسوية مخزون عند الحاجة.

## 9. الجرد
1. افتح شاشة الجرد.
2. أنشئ جلسة جرد.
3. سجل الكميات الفعلية.
4. اعتمد الجرد بعد الانتهاء.
5. النظام سيعدل المخزون حسب الفرق.

## 10. التقارير
تعرض شاشة التقارير:
- إجمالي المبيعات.
- المرتجعات.
- الأرباح.
- المصروفات.
- المنتجات الأكثر مبيعًا.
- العملاء الأعلى شراءً.
- المنتجات منخفضة المخزون.

## 11. النسخ الاحتياطي
يفضل عمل Backup دوريًا من شاشة الإعدادات أو النسخ الاحتياطي.
